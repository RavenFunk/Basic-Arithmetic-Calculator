

# Basic Arithmetic Calculator
Created: 7/20/2026
Author: Raven Funk


My first ever project in Python.


## How to run:

 - Ensure you have already installed Python.
```console
python main.py
```

## Capabilities:
This basic arithmetic calculator is capable of the following:
 - Addition
 - Subtraction
 - Multiplication
 - Division
 - Continuous calculation
 - "new" keyword


## Example Usage:
```text
> 10 + 5
5

> * 2
30

> new

> 90 + 10
100

> 
```
